What is KNN?

In pattern recognition, the k-nearest neighbors algorithm (k-NN) is a non-parametric method used for classification and regression. In both cases, the input consists of the k closest training examples in the feature space. The output depends on whether k-NN is used for classification or regression:

  • In k-NN classification, the output is a class membership. An object is classified by a majority vote of its neighbors, with the object being assigned to the class most common among its k nearest neighbors (k is a positive integer, typically small). If k = 1, then the object is simply assigned to the class of that single nearest neighbor.
  • In k-NN regression, the output is the property value for the object. This value is the average of the values of its k nearest neighbors.

In this web app, we explore KNN classification using images, which are innately in a high dimension. If we represent each pixel with a dimension, then each image has over 10,000 dimensions! My goal was to see whether Python could compute nearest neighbors in that high of a dimension efficiently, and as it turned out, it can (although this webapp may take longer time than usual to refresh). In this setting specifically, I wanted to see how well the algorithm could distinguish between images of people wearing glasses, and images of people who aren't - whether there was enough difference between those two groups in the 10,304D feature space. Click on the people on the left and adjust the number of neighbors to see how well the program does! Otherwise, you can read up more on KNN and loads of other fun algorithms in An Introduction to Statistical Learning (ISLR) by Gareth James et al. (free online!).

How does it work?

The script uses a training set of 40 men and women, wearing glasses or no glasses. Each person contributed 10 images of themselves shot in different angles, so in reality, there are 400 samples, each with 10,304 features (each image was 112x92). It turns out it's pretty easy to use Sci-Kit Learn in Python to train all sorts of models - the hardest part really was figuring out how to structure the data with Panda dataframes. The people on the left were taken out of the training set, the KNN model was fit and trained, and finally it was used to predict the class of the test subject. One caveat: it's worth mentioning that in the set of people, only a handful of the 40 were actually wearing glasses. This would skew the classification in favor of better classifying people without glasses; if I could, I'd choose a more balanced set. Going along with that, in the future I'd also like to test the effects of dimensionality reduction with PCA on the high-dimensional feature space. Performing KNN "upstairs" wasn't too much of an issue - in fact, it led to some perfect classifiers - but it'd be interesting to see what would happen in a lower dimension.
